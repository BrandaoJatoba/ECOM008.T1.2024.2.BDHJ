typedef struct huffman_code huffman_code

Representa um código de Huffman em bits.

Os bits são armazenados em um inteiro (até 32 bits). Como geralmente apenas 1 a 18 bits são utilizados, é necessário informar quantos bits são válidos.

◆ compress()

int compress ( const char * file_name)

Função principal de compressão de arquivo.

Etapas:

  1. Gera o nome do arquivo de saída com extensão .huff;
  2. Abre o arquivo original e calcula a frequência dos bytes;
  3. Cria lista encadeada com os nós baseados na frequência;
  4. Gera a árvore de Huffman;
  5. Constrói o dicionário de códigos;
  6. Escreve o arquivo comprimido;
  7. Libera toda a memória alocada dinamicamente.
Parameters
file_nameNome do arquivo a ser comprimido.
Returns
0 em caso de sucesso.

int write_compressed_file ( FILE * uncompressed_file,
char * compressed_file_name,
void * ht,
void * dictionary[] )

Escreve o arquivo comprimido utilizando a árvore e o dicionário.

A função usa buffers de leitura e escrita:

  • O buffer de leitura obtém os bytes do arquivo original.
  • O buffer de escrita armazena os bits dos códigos de Huffman.

Operações bitwise garantem que somente um byte seja escrito por vez. Qualquer sobra de bits no buffer é processada no próximo ciclo.

Também grava a árvore Huffman serializada e o tamanho do "lixo" ao final do processo no cabeçalho do arquivo comprimido.

Por fim, imprime um relatório com estatísticas da compressão.
