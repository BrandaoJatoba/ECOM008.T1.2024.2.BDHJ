◆ free_nodes()

void free_nodes ( void * node,
int mode )

Libera os nós da árvore ou da lista encadeada de Huffman.

A função suporta dois modos distintos de liberação de memória, considerando que a estrutura de dados é mista: os nós podem representar tanto elementos de uma lista encadeada quanto de uma árvore binária.

O parâmetro mode define o comportamento:

  • 0: trata os nós como uma lista encadeada.
  • 1: trata os nós como uma árvore binária.

Essa flexibilidade permite a conversão natural da lista de frequências em uma árvore de Huffman, reaproveitando a estrutura.

Parameters
nodePonteiro para o nó inicial (lista) ou raiz (árvore).
modeModo de liberação: 0 para lista, 1 para árvore.
